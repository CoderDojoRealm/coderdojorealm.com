Scratch

Recommended for beginner to intermediate coders.

With Scratch, you can program your own interactive stories, games, and animations — and share your creations with others in the online community.

Scratch helps young people learn to think creatively, reason systematically, and work collaboratively — essential skills for life in the 21st century.

Getting Started

If you're new to Scratch and don't know where to start, follow this guide:

  1. Visit the scratch website and register for a new account (it's free).
  2. Download and print the Beginner Tutorial below
  3. Watch the Getting Started video from the Introductory Videos section below

Challenges

Try these challenges.

  • Make a ball bounce
  • Draw a flower using the pen commands
  • Build a game where two players try to shoot each other
  • Use real photos of yourself and make yourself dance to music on screen
  • Recreate a classic 1980s arcade game such as Donkey Kong, Frogger, Asteroids, Pac Man, etc.
  • Connect scratch to our LEGO Mindstorm robotics kit
  • Create a simulation of the solar system (planets.zip)
