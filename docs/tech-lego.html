LEGO Mindstorms

CoderDojo Realm is fortunate to have a LEGO Mindstorms kit, to allow ninjas to experiment with robotics.

The Mindstorms kit provides a range of sensors and motors to build and code robots, and it can also connect to Scratch for more interesting and complex coding projects to be built.
